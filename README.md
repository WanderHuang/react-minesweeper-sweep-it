## 扫雷游戏

#### 规则

1. 在n * n的矩阵内，找出所有的地雷
2. 鼠标左击表示翻开，右击表示标记或取消标记
3. 翻开方块时，如果是地雷，游戏结束
4. 每个方块中的数字表示周围一圈（八个）九宫格内，存在的地雷总数
5. 翻开空白方块时，自动翻开周围方块
